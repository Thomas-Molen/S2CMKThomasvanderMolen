SPRINT 3
	Deze sprint ben ik begonnen met het werken aan mijn websysteem, dit systeem bestaad uit een leaderboard waar runs vanuit het spel getoond kunnen worden,
	samen met een gedeelte data management. Verder heb ik ook de eerste poging tot de Circus trein opdracht gedaan, deze zal ik de volgende sprint verbeteren
	op basis van feedback die ik heb ontvangen.

Web Functionalitiet:
	- Web opgezet via laravel.
	- Registreer en login werkend.
	- Database opgezet met alle benodigde velden.
	- Alle data in de vorm van CRUD's gemaakt met de benodigde overzicht, edit en inzicht pagina's.
	- Leaderboard pagina gemaakt die de top 100 runs op volgorde laat zien.

Onderdelen buiten Web:
	- Trello planning.
	- ERD aanpassingen.
	- Klassen diagram aanpassingen.
	- Game aanpassingen voor betere OOP en minder dependencies.
	- Circus trein workshop v1 gemaakt met flowcharts en klassen diagrams.
